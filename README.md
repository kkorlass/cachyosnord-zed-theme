# cachyosnord-zed-theme
A zed theme based on the CachyOS Nord Plasma color scheme

## Installation
```
git clone https://github.com/kkorlass/cachyosnord-zed-theme.git
cd cachyosnord-zed-theme
cp ./CachyOS\ Nord.json ~/.config/zed/themes/CachyOS\ Nord.json
```
In Zed press Ctrl-K Ctrl-T and apply theme "CachyOS Nord"
